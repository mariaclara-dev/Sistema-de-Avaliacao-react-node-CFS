# Sistema de Avaliação 

Projeto Full Stack desenvolvido em React e Node.js.

## Objetivo
Permitir que usuários realizem avaliações com notas e comentários, exibindo a média geral.

## Tecnologias
- React
- Node.js
- Express
- Axios

## Funcionalidades
- Cadastro de avaliações
- Listagem de avaliações
- Cálculo da média
- Ranking de notas

## Como executar

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
